# DriveFleet 

A premium full-stack car rental platform built with Next.js, Firebase, MongoDB, Express.js and Tailwind CSS.

#  Features

- Firebase Authentication
- Google Login
- Private Routes
- Add Car
- Explore Cars
- Car Details Page
- Book Now System
- My Bookings
- My Added Cars
- Update Car
- Delete Car
- Cancel Booking
- Search & Filter
- JWT Authentication
- Responsive Premium UI
- MongoDB Database
- Protected Dashboard

#  Installation

## Client

```bash
git clone CLIENT_REPO_LINK
cd drivefleet-client
npm install
npm run dev
```

## Server

```bash
git clone SERVER_REPO_LINK
cd drivefleet-server
npm install
npm run dev


---
```
#  Environment Variables

## Client `.env.local`

```env```
NEXT_PUBLIC_API_URL=https://drivefleet-server-fv0g.onrender.com
```

## Server .env

env
PORT=5000
MONGODB_URI=your_mongodb_uri
JWT_SECRET=drivefleet_secret_key
CLIENT_URL=http://localhost:3000
```
